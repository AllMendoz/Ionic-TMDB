Información Resultados

Esta aplicación fue realizada como prueba técnica para la postulación al cargo de Programador en Parquesoft Tuluá.


La App resultante, consta de la generación de un catálogo de consulta de películas, series y personalidades bajo el framework Ionic (https://ionicframework.com/) en su última versión, usando como backend la API TheMovieDb (https://www.themoviedb.org/), resolviendo los inconvenientes que posee la API al no generar un search o consulta con alguna especificación de sort u ordenamiento.
